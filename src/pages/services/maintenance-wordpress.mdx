Les sites réalisés avec le CMS **WordPress** sont soumis régulièrement à des attaques. Des robots (scripts) tentent de rentrer en exploitant les failles des plugins ou du cœur de **WordPress**.
**WordPress** étant un CMS majoritaire sur le web est logiquement plus attaqué que les autres systèmes.

Pour vous aidez à préserver votre site web et votre réputation, nous vous proposons un service de maintenance des sites réalisés avec le CMS WordPress.

Notre service comporte 3 offres :

## Maintenance WordPress préventive

### Mise à jour WordPress

Pour limiter les risques de faille, il est nécessaire d'effectuer les **mises à jour régulièrement**.
En effet, les plugins et le cœur de WordPress sortent régulièrement des mises à jour pour corriger des **failles de sécurité**. Si vous n'effectuez pas les mises à jour régulièrement, vous vous exposez à un risque potentiel d'intrusion.

### Sauvegarde

Il est également nécessaire d'effectuer des **sauvegardes** de votre site et de la base de donnée.
En cas de **piratage** de votre site, il sera toujours possible de reconstruire le site avec une version non piraté.

### Sécurité

Utiliser un **plugin de sécurité** permet de bloquer les attaques malveillantes et les tentatives d'intrusions.
Il permet de scanner les fichiers et de détecter des changements anormaux de fichiers.

Notre offre maintenance :

- Mise à jour des plugins et du coeur de WordPress.
- Sauvegardes régulieres du site et de la base de donnée.
- Configuration d'un plugin de sécurité.

### Forfait maintenance WordPress

Tous les forfaits nécessitent un forfait "prise en main" de **400€ HT** pour la mise en place de la maintenance mensuel.
Dans cette étape, nous effectuons un état des lieux de votre site et nous mettons en place quelques bonnes pratiques :

- Réglages de base pour le cache.
- Installation et configuration d'un plugin de cache.
- Installation et configuration d'un plugin de sécurité
- Installation et configuration de la sauvegarde

## Optimisation des performances de votre site WordPress

La **vitesse de chargement** est devenue un critère important pour le **SEO** (optimisation pour les moteurs de recherche). Mais également pour l'expérience des utilisateurs.
Un internaute quitte plus rapidement un site lent. Et c'est encore plus vrai depuis que l'utilisation sur mobile est passée devant l'utilisation sur ordinateur.

Nous vous proposons une offre pour optimiser les performances de votre site.
Les étapes de cette prestation :

### Audit du site :

Point sur l'état de votre site et compte rendu sur les points à optimiser.

**A ce niveau, il est possible qu'il soit impossible d'optimiser votre site. Dans ce cas, rien ne vous sera facturé**

### Réalisation des optimisations :

Application des points d'optimisations. Les optimisations sont réalisées une par une avec comparaison de l'amélioration des performances.

### Rapport
