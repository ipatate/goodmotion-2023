Développeur web depuis plus de 15 ans, j'ai accumulé une grande expérience dans la conception et le développement d’applications web et de sites web.

J'ai fait mes premières armes avec PHP, from scratch au départ puis avec Symfony dès la version 1.4, en passant par du Prestashop.

En 2015, je me suis spécialisé dans les applications Web JavaScript et j'ai rapidement adopté React.js dès le début pour construire mes premières applications web.

À partir de 2018, j'ai été séduit par les générateurs de sites statiques. J'ai rapidement pris plaisir à utiliser Gatsby JS, Next.js et Nuxt.js.

Désormais, je dispose d'une multitude d'outils pour développer. J'ai la possibilité de choisir l'outil de plus adapté.

Cela peut être un site sous WordPress ou avec Craft CMS pour certains clients. Ou bien une web application hybride avec Next.js.

Les clients peuvent se reposer entièrement sur mon expérience et j'aime avoir une relation de confiance sur le long terme.
